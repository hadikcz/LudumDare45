# About game:
You are poor farmer, who lose everything, and you must start again. Plant a seed, harvest it, create flour and finally bread for your family.

# How to play ? / Controls
- W - Jump
- A - Move left
- D - Move right

- Q - Drop item
- E - Pickup/interact
